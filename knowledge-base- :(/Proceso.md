# Proceso

## 1. Contacto inicial
El cliente se comunica con nosotros a través de la web, WhatsApp o formulario. Se recopila información básica del proyecto.

## 2. Evaluación
Analizamos la información y, si es necesario, realizamos una visita técnica para entender el estado de la instalación.

## 3. Cotización
Se presenta una propuesta clara con alcance, tiempos y costos estimados.

## 4. Ejecución
Se realiza el trabajo técnico según el servicio requerido (instalación, certificación, asesoría, etc.).

## 5. Seguimiento
Se verifica el cumplimiento del servicio y se brinda acompañamiento adicional si es necesario.
